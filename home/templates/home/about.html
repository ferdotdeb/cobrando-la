Acerca de Cobrando.la

Estamos dedicados a simplificar y democratizar el acceso a las transferencias bancarias en México.

Nuestra misión

Creemos que compartir información bancaria no debería ser complicado, inseguro, ni inaccesible.

🎯

Accesibilidad financiera para todos

Cobrando.la nace de la necesidad personal de facilitar el intercambio de datos bancarios de manera segura, verificada y accesible. Muchas personas enfrentan barreras al compartir su información de pago: errores al transcribir números largos, falta de validación inmediata, o simplemente la incomodidad de compartir datos sensibles por mensajes de texto.

Nuestro objetivo es ofrecer una plataforma donde cualquier persona pueda generar un perfil público con sus datos bancarios verificados, controlando en todo momento qué información se comparte y con quién, eliminando fricciones y aumentando la confianza en las transacciones digitales.

🔐

Seguridad y privacidad

Proteger tus datos es nuestra prioridad. Implementamos validaciones automáticas y te damos control total sobre qué información compartes.

✨

Simplicidad

Una interfaz clara y directa. Sin complicaciones innecesarias, sin pasos confusos. Solo lo esencial para que compartas tus datos de forma rápida y segura.

Tecnologías y stack

Construido con tecnologías modernas, robustas y probadas en producción.

🐍

Django

Framework web de Python para desarrollo rápido, seguro y escalable. Backend robusto con ORM integrado.

🎨

Tailwind CSS

Framework CSS utility-first para diseños modernos, responsivos y altamente personalizables.

🗄️

Base de datos

SQLite para desarrollo local, PostgreSQL recomendado para producción. Almacenamiento seguro y eficiente.

🐳

Docker

Contenedores para desarrollo y despliegue. Configuración reproducible y fácil de compartir.

💻

GitHub

Control de versiones, colaboración y CI/CD. Todo el código disponible públicamente para la comunidad.

✅

Validaciones

Algoritmo de Luhn para tarjetas, verificación de CLABE, validación de bancos mexicanos registrados.

📦

Arquitectura modular

Cobrando.la está estructurado en aplicaciones Django independientes: accounts para gestión de usuarios, bank_details para datos bancarios, home para páginas públicas. Esto facilita el mantenimiento, las pruebas y la escalabilidad del proyecto.
